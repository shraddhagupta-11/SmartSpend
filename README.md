# SmartSpend 💰

A full-stack fintech-inspired personal finance management application built with Django.

## Features

- User Authentication
- Expense Tracking
- Income Management
- Budget Planning
- Savings Goals
- Financial Analytics Dashboard
- Interactive Charts
- Responsive UI
- Profile Management

## Tech Stack

- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript
- Chart.js
- Bootstrap Icons

## Installation Guide

```bash
git clone https://github.com/shraddhagupta-11/SmartSpend.git

cd SmartSpend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
